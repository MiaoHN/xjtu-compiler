# 语法分析

本次实验使用 `yacc` 构建一个语法分析器

## yacc 的使用

`yacc` 是一个编译器代码生成器，常用来进行语法分析。其输入是**BNF 范式**，输出的是基于表驱动的编译器，包含输入的语法规约的处理代码部分，采用**LALR(1) 语法分析**方法，可以配合词法分析器 Lex 产生

## 基于语法分析的计算器的实现

### 实验目标

本次实验通过 lex 和 yacc 组合套件实现一个四则运算器

- 数字：`[0-9]+(\.([0-9]+)?)?`
- 其他符号：`+-*/()`
