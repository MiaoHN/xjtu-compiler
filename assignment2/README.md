# Cool 语言堆栈机

## 1. 操作说明

| 命令  |              含义              |
| :---: | :----------------------------: |
| `int` |       将整数数添加到栈中       |
|  `+`  |          栈中压入 `+`          |
|  `s`  |          栈中压入 `s`          |
|  `e`  | 计算栈顶表达式的值（详见下文） |
|  `d`  |          打印栈中内容          |
|  `x`  |              退出              |

`e` 命令的作用：

1. 如果 `+` 在栈顶，那么 `+` 出栈，栈顶两整数弹出并相加，将结果压入栈
2. 如果 `s` 在栈顶，那么 `s` 出栈，下面两个选项在栈内交换
